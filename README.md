# tf-viewer
A self-hosted GraphQL API for interfacing with GPS-based activity files.

*Current status: Rewriting, alpha release soon*

## Features
- Multi-user support
- Easy setup and deployment
- OAuth2-based authorization
- Supported file formats:
  - [x] FIT
  - [ ] GPX
  - [ ] TCX
